## Wireframe 1: Dashboard (Main Screen)

See full wireframe in previous section. Key areas:

1. **Global Navigation** (top, sticky)
2. **Hero Metrics** (6 KPI cards in 3x2 grid)
3. **Action Items** (alert-style cards)
4. **Campaign Cards** (3-column grid)
5. **Activity Feed** (chronological list)
6. **Tips Section** (collapsible)

**Responsive Behavior:**
- Desktop (1440px): 3-column layout for campaigns
- Tablet (768px): 2-column layout for campaigns
- Mobile (375px): Single column, metrics become swipeable carousel

## State Variations

### Button States
- **Normal**: Default color
- **Hover**: Slightly darker, shadow
- **Active/Pressed**: Darker, inset shadow
- **Disabled**: 50% opacity, no pointer
- **Loading**: Spinner inside button, disabled

### Input States
- **Normal**: Border gray-300
- **Focus**: Border blue, shadow
- **Error**: Border red, red helper text
- **Success**: Border green (optional)
- **Disabled**: Gray background, no pointer

### Card States
- **Normal**: White background, subtle shadow
- **Hover**: Elevated shadow (interactive cards)
- **Selected**: Blue border
- **Error**: Red border
- **Loading**: Skeleton placeholder
